Class CortexCredentialProvider<T>

Abstract class to provide credentials for multiple datalakes. If you want to extend this class then you must implement its storage-related methods. T describes the type of the optional metadata that can be attached to any datalake's record.

Type parameters

  • T

    type of the metadata to attach to any data lake instance
